# Spring-Legacy-Project-BOARD
스프링 레거시 + JSP를 이용한 간단한 게시판


## 🖥️ 프로젝트 소개
연습삼아 만들어본 간단한 CRUD 게시판입니다!
<br>

### ⚙️ 개발 환경
- `Java 11`
- `JDK 11.0.19`
- **IDE** : IntelliJ IDEA
- **Framework** : Spring Framework 5.3.20
- **Database** : MySQL 8.0.22
- **ORM** : Mybatis
- **Web-Server** : Apache Tomcat 9.0.65
